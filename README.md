# 美本工程国际生选校全案系统

> **留学择校 Vision v1.0** — 面向中国大陆籍国际生的美本工程选校全案工具

## 这是什么

一个 Claude Code Skill，专为**中国大陆籍本科工程申请者**设计。打破唯排名论的选校误区，以「就业创业资源」为核心权重，整合官方录取数据 + 中外学生社区真实案例 + 招生官公开反馈，输出个性化选校方案。

## 核心能力

| 模块 | 说明 |
|---|---|
| **背景竞争力评估** | 对标大陆籍工程申请池，定位优势短板 |
| **分层选校清单** | Reach/Match/Safety 各 6-8 所，含录取数据 + 预算匹配 |
| **单校深度分析** | CDS 录取数据 + Reddit/CC/小红书真实案例 + 就业创业评分 |
| **分区域文书策略** | UC系/东北私立/中部公立/南部产业校，含招生官偏好 + 避坑红线 |
| **差异化破局** | 方向避卷、地域错配、人设差异化、Demonstrated Interest 强化 |
| **CC转UC专项** | TAG协议、Feeder CC、风险提示、成本对比 |
| **就业创业权重** | 产业区位 > 校企合作 > 创业支持 > 校友生态 > 政策友好度 |

## 专业范围

✅ 计算机工程 CE、电子工程 EE、机械工程 ME、航空航天 AE、土木工程、材料工程、机器人工程、工程物理、能源工程、工程管理 等

❌ 化学工程、生物工程、生物医学工程（严格排除）

## 数据源与权重

- **60%** 官方权威：College Board CDS、院校工程学院官网、US News 工程专项排名、招生官博客/AMA
- **25%** 真实案例：Reddit r/ApplyingToCollege、College Confidential、小红书、CUUS
- **15%** 趋势补充：Niche、X/Twitter 行业 KOL、产业报告

## 安装

将 `SKILL.md` 放到 `~/.claude/skills/cn-intl-us-engineering-college-advisor/` 目录下，重启 Claude Code 即可。

触发关键词：`美本工程选校` `工科择校` `大陆生美本工程申请` `CC转UC工程` `工程选校就业导向`

## 声明

本工具提供的是**数据驱动的决策辅助**，不构成唯一择校依据。录取结果受多种因素影响，请结合官方最新信息和个人判断使用。
